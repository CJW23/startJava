## Oracle JDK와 OpenJDK

- Java 애플리케이션을 실행하기 위해서는 JVM이 필요하고 컴파일하기 위해서는 JDK가 필요하다. 일반적으로 JDK를 설치하면 JVM(Hotspot이라고도 표현, Java 기술의 핵심)도 함께 설치된다.

- JDK는 2개 버전으로 나뉜다. 하나는 폐쇄적인 상업 코드 기반의 `Oracle JDK`이고 하나는 오픈 소스 기반의 `OpenJDK` 이다.
